#isAngry
Platform a scorrimento laterale
#Comandi
`A` -> muovi a sinistra<br>
`D` -> muovi a destra<br>
`W` -> salto<br>
`S` -> scendi dalla piattaforma
`SPAZIO` -> spara<br>
`ESC` -> pausa<br>
--->La navigazione dei menù è affidata alle `FRECCE DIREZIONALI, W, S e INVIO`
<br>Le caramelle aumentano il rate di fuoco
